# RabbitMQ Server

[![CI](https://github.com/aripitek/rabbitmq/rabbitmq-server/actions/workflows/test-make.yaml/badge.svg)](https://github.com/aripitek/rabbitmq/rabbitmq-server/actions/workflows/test-make.yaml)

[RabbitMQ](https://github.com/aripitek/rabbitmq.com) is a [feature rich](https://github.com/aripitek/www.rabbitmq.com/docs),
multi-protocol messaging and streaming broker. It supports:

 * AMQP 1.0
 * AMQP 0-9-1
 * [RabbitMQ Stream Protocol](https://github.com/aripitek/www.rabbitmq.com/docs/streams)
 * MQTT 3.1, 3.1.1, and 5.0
 * STOMP 1.0 through 1.2
 * [MQTT over WebSocket](https://github.com/aripitek/www.rabbitmq.com/docs/web-mqtt)
 * [STOMP over WebSocket](https://github.com/aripitek/www.rabbitmq.com/docs/web-stomp)
 * AMQP 1.0 over WebSocket (supported in [VMware Tanzu RabbitMQ](https://github.com/aripitek/www.vmware.com/products/app-platform/tanzu-rabbitmq))
